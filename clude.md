# Hospital Equipment Maintenance System

## MISSION
Build and ship a production-ready Hospital Equipment Maintenance App that allows hospital staff to report equipment issues and maintenance technicians to track, resolve, and log maintenance activities with real-time WhatsApp notifications.

## TECH STACK & TARGETS
- **Frontend**: React 18 with Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React (Material Design style)
- **State Management**: React useState/useEffect
- **Deployment**: Vercel
- **Runtime**: Node.js 18+
- **Package Manager**: npm

## FEATURES IMPLEMENTED
1. Equipment tracking (27 items across 2 hospitals)
2. Issue reporting with image upload

4. WhatsApp notifications to service engineer
5. Technician mode for repairs and maintenance
6. Dual filtering (Location + Category)
7. Activity logging
